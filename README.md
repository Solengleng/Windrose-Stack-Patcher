### Troubleshooting

**Game path not found?**
- Click the Browse button and select the folder containing `R5.exe` manually.
- Make sure you are using the Steam version of the game.

**Mod not working?**
- Make sure no other mods are conflicting with inventory files.
- Delete the `~mods` folder and try applying again.
- Run the patcher as Administrator.

**Need to remove the mod?**
- Click the red **Remove all mods** button.
